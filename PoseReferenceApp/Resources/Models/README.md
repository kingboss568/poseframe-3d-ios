PoseFrame Studio character model bundle

Place converted Rocketbox USDZ files here before App Store release:

- Free models: `Free/<usdzName>.usdz`
- Pro models: `Pro/<usdzName>.usdz`

The app data model stores the `usdzName` for all 8 characters. The current
SceneKit renderer keeps a procedural realistic fallback so local builds remain
usable while final Rocketbox assets are being converted.
